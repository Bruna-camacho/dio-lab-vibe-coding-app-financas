# 💸 MetaSmart - App de Organização de Finanças Pessoais com Vibe Coding

Este projeto foi desenvolvido como um Desafio de Projeto da DIO de Vibe Coding utilizando o Lovable e o Copilot Web. O foco é desenvolver o conceito de um **App de Organização de Finanças Pessoais**, mas, acima de tudo, aprender o **jeito Vibe de programar com IA**.


## 🪄 PRD Refinado no Copilot Web

```
**# PRD – App de Organização Financeira MetaSmart com Conversa Natural**

**## Visão Geral**
O MetaSmart será um aplicativo de finanças pessoais baseado em conversas em linguagem natural, totalmente focado na criação e gerenciamento de metas financeiras.
A proposta é permitir que o usuário descreva suas intenções de forma simples, como:

- "Quero juntar R$ 2000 para comprar um celular."
- "Estou gastando muito em delivery."
- "Aumente minha meta de economia para este mês."

A IA será responsável por interpretar as informações, criar metas, ajustar valores, acompanhar o progresso e oferecer recomendações automáticas.

**## Problema a Resolver**
A maioria dos aplicativos financeiros exige preenchimento manual, metas rígidas, muitas telas e pouca personalização.
Por causa disso, muitos usuários abandonam o controle financeiro rapidamente.

O MetaSmart propõe uma solução baseada em:

- Conversas naturais
- Metas dinâmicas
- Ajustes automáticos conforme o comportamento do usuário
- Relatórios simplificados e acessíveis

**## Público-Alvo**
Pessoas que desejam organizar suas finanças de forma prática e fácil, especialmente aquelas que:

- Não têm costume de usar apps financeiros
- Buscam metas simples e intuitivas
- Gostariam de recomendações personalizadas
- Preferem interagir por linguagem natural
- Querem acompanhar o progresso sem complexidade

**## Funcionalidades-Chave**
1. Criação de metas via chat:
   O usuário descreve suas metas em linguagem natural, e a IA converte em metas estruturadas.

2. Acompanhamento automático de metas:
   A IA monitora o progresso e faz projeções com base nos hábitos reais do usuário.

3. Recomendações inteligentes:
   A IA sugere ajustes e fornece dicas práticas para atingir metas mais rapidamente.

4. Registro de gastos por conversa:
   Exemplo: "Gastei R$ 45 com comida hoje."

5. Painel de progresso:
   Exibe gráficos simples e objetivos mostrando a evolução das metas.

6. Ajuste automático baseado no comportamento:
   Se o usuário gasta mais do que o previsto, o app recalcula metas e prazos.

7. Agente Financeiro MetaSmart:
   Personagem conversacional que orienta e motiva o usuário.

**## Princípios de Design Universal**
O MetaSmart seguirá os princípios de Design Universal para garantir acessibilidade e usabilidade para todas as pessoas.
Isso inclui:

- Telas limpas e legíveis
- Navegação simples
- Linguagem clara
- Compatibilidade com leitores de tela
- Redução de sobrecarga cognitiva
- Feedback visual e auditivo

*## Entregáveis Esperados da IA*
A IA deve gerar um plano de MVP contendo:

- Telas principais: chat, metas, progresso e relatórios
- Estrutura do fluxo do usuário
- Requisitos técnicos: NLP, motor de metas, categorização automática
- Estratégia de validação inicial com usuários reais
- Recomendações de design acessível
- Protótipo simples baseado neste PRD
```

**Reflexões sobre o Projeto MetaSmart**
**O que funcionou bem?**
A integração entre o PRD refinado e o Lovable foi um ponto forte. Ter um documento claro ajudou o gerador a estruturar as telas de forma consistente, e isso facilitou muito a visualização do fluxo do app. Além disso, os testes com as telas mostraram que a ideia de usar linguagem natural realmente torna o protótipo mais acessível e intuitivo.

**O que não funcionou como o esperado?**
O processo de geração das telas exigiu alguns ajustes no prompt. No início, o Lovable interpretava apenas parte das instruções e criava telas incompletas. Foi preciso adaptar a forma de escrever os comandos para que o sistema entendesse melhor. Isso mostrou que nem sempre basta ter um PRD detalhado — é necessário traduzi-lo em instruções diretas para a ferramenta.

**O que aprendi sobre conversar com IAs?**
Aprendi que clareza e objetividade são fundamentais. Quanto mais estruturada e específica é a comunicação, melhor o resultado. Também percebi que a IA responde bem quando recebe contexto estratégico (como o PRD) seguido de instruções práticas. É quase como ensinar alguém: primeiro explicar a visão geral, depois dar as tarefas passo a passo.
